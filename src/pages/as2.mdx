# Advancing the standard in B2B Messaging


- Privacy
- Authentication
- Nonrepudiation of origin and receipt of the message
- Data integrity

The AS2 application provides a medium to exchange business data with partners by
configuring an account in webMethods.io Integration. The application supports
the AS2 protocol versions 1.1 and 1.2, and our open version of AS 2.0 ( AS2^2 )

## Payload Types 

| **Key**     | **Value**                                                               |
| ----------- | ----------------------------------------------------------------------- |
| data        | Document Payload you want to send\.                                     |
| stream      | Object java\.io\.InputStream object you want map from EDI or XML data\. |
| contentType | String Content type to assign to an outbound message\.                  |
<br />

#### The following options are available by default:

application/edi-x12
application/edifact
application/xml
application/json
text/binary
